## Selected Projects

### **[Coevolutionary ARL Agent vs EGT Population Research](https://github.com/Srithwak/Coevolutionary_ARL_agent_vs_EGT_population_research)**
Simulating competitive dynamics in stock-market environments using adversarial RL and evolutionary game theory to answer the question: "Can a single ARL agent consistently outperform and exploit a heterogeneous population of EGT agents evolving via Replicator Dynamics?"
